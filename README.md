# full-stack-experiment-3
his project is a Full Stack Experiments Dashboard that demonstrates basic web development concepts using HTML, CSS, and JavaScript.
It includes three interactive modules:

Product Card
Library Management System
Person Hierarchy (OOP Concept)

The project is designed for learning purposes and showcases UI design, DOM manipulation, and object-oriented programming in JavaScript.

🚀 Features
1.  Product Card
Displays a product with price and stock status
Toggle between In Stock / Out of Stock dynamically
2.  Library Management
Add new books with author name
Remove books from the list
Search books dynamically
Real-time UI updates
3.  Person Hierarchy
Demonstrates OOP concepts (Inheritance)
Classes used: Person, Student, Teacher
Displays different types of people with details
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)

 How to Run
Download or clone the project
Open index.html in any browser
Click on different cards to explore features
 Learning Outcomes

After completing this project, you will be able to:

Understand DOM manipulation using JavaScript
Implement event handling (click, input, etc.)
Build interactive UI components using HTML & CSS
Apply Object-Oriented Programming (OOP) concepts in JavaScript
Create simple dynamic web applications without any framework
